45 solving techniques

The Ukodus engine implements the most comprehensive set of human-style Sudoku solving techniques of any browser-based Sudoku app. Every puzzle is rated by the hardest technique required to solve it without guessing, using the Sudoku Explainer (SE) difficulty scale.

The 45 techniques are organized into 10 families: Singles, Pairs & Triples, Intersections, Fish (from X-Wing to Kraken Fish), Wings, Chains, Rectangles, Almost Locked Sets (ALS), Forcing Chains, and advanced patterns like Sue de Coq and Death Blossom.

View the complete technique catalog with SE ratings.

Hints that teach you

When you request a hint, Ukodus doesn't just reveal the answer. It shows you the solving technique in three progressive steps:

  • Step 1: The technique name (e.g. "X-Wing on digit 4")
  • Step 2: The involved cells highlighted on the board
  • Step 3: A full logical proof showing exactly why the elimination or placement is valid

This means every hint is a learning opportunity. Over time, you'll recognize these patterns yourself and need fewer hints.

8 difficulty tiers

Ukodus offers 8 difficulty tiers, each defined by the solving techniques required — not by how many clues are given:

  • Beginner (SE 1.5-2.0) — Hidden singles only
  • Easy (SE 2.0-2.5) — Naked singles
  • Medium (SE 2.5-3.4) — Pairs and intersections
  • Intermediate (SE 3.4-3.8) — Hidden triples, basic fish
  • Hard (SE 3.8-4.5) — Box/line reduction, wings
  • Expert (SE 4.5-5.5) — Fish, rectangles, advanced wings
  • Master (SE 5.5-7.0) — Chains, ALS, Medusa (secret tier)
  • Extreme (SE 7.0-11.0) — Forcing chains, backtracking (secret tier)

The Sudoku Galaxy

The Galaxy is a unique feature: an interactive force-directed graph where every puzzle ever played on Ukodus appears as a star. Puzzles cluster by the techniques they share, with 10 families represented by different colors. New puzzles appear in real time via WebSocket. Zoom, pan, filter by technique, and click any star to see its profile and play it.

Built with Rust + WebAssembly

The engine is written in Rust and compiles to WebAssembly for the browser, native code for iOS (via UniFFI), and a terminal UI. The same solver, generator, and technique engine powers all three platforms — if you share a puzzle code between web and iOS, it's the exact same puzzle solved by the exact same engine.

The full source is available at github.com/kcirtapfromspace/sudoku under the MIT license.

Frequently asked questions

Is Ukodus really free?

Yes. No ads, no premium tier, no in-app purchases. The project is open source and funded by the developer.

Do I need to create an account?

No. Just open ukodus.now/play and start playing. Your progress saves to your browser's localStorage. For leaderboards, you pick a 3-6 character tag (like "ACE") — no email or password.

What makes the difficulty rating accurate?

Ukodus rates puzzles using the Sudoku Explainer (SE) scale, the community standard used by competitive Sudoku enthusiasts. The engine solves each puzzle using only human-style logic (no brute force), and the SE rating equals the hardest technique required. A puzzle rated SE 4.6 genuinely requires techniques like Unique Rectangle or XYZ-Wing.

How does the hint system work?

Press ? for a hint. The first press shows the technique name. Press again to see the involved cells highlighted. Press again for the full logical proof. Press ! to apply the hint to the board. Every hint uses the same 45-technique engine that rates the puzzle.

Can I share puzzles?

Yes. Click Share to get an 8-character code (like M1A2B3C4) or a URL. The code works across the web app, iOS app, and terminal UI — same engine, same puzzle.

What are the "secret" difficulty tiers?

Master (SE 5.5-7.0) and Extreme (SE 7.0-11.0) are hidden by default. They require advanced techniques like forcing chains, ALS chains, and 3D Medusa. They can be unlocked in the difficulty selection menu.

Is there an iOS app?

Yes. The iOS app uses the same Rust engine compiled to native code via UniFFI. It supports Game Center leaderboards and iCloud sync. Share puzzle codes between web and iOS.

Does Ukodus track me?

No. There are no analytics, no advertising trackers, no cookies. The only data sent to the server is anonymous gameplay results (time, difficulty, technique profile) for the Galaxy visualization and leaderboards. A randomly generated player ID is stored in your browser's localStorage. See our privacy policy.
