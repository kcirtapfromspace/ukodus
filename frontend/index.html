How it works

The Rust engine rates each puzzle using 45 human-style solving techniques, producing a precise SE (Sudoku Explainer) rating. The Galaxy API collects anonymous play results and builds the constellation in real time.

Play a puzzle

Start a game at any difficulty tier. The WASM engine generates a unique puzzle with a guaranteed single solution.

Results flow in

When you finish, anonymous play data (time, difficulty, technique profile) is sent to the Galaxy API. No accounts needed.

Galaxy grows

New puzzles appear as stars, connected to similar puzzles. Live WebSocket updates pulse new nodes into the visualization.
