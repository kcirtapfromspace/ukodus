Difficulty Levels

Every Ukodus puzzle is rated using two systems: a technique-based difficulty tier and a numerical Sudoku Explainer (SE) rating. The SE rating measures the hardest technique needed to solve the puzzle without guessing.

6 Difficulty Tiers

From simple scanning to deep logical chains. Each tier represents a step up in the reasoning required.

Beginner

SE 1.5 – 2.0
Hidden singles

Simple scanning. Each step has an obvious answer. Look for rows, columns, or boxes where only one cell can hold a particular digit.

Easy

SE 2.0 – 2.5
Naked singles

Direct candidates. Look for cells with only one possibility. If you can count to nine, you can solve these.

Medium

SE 2.5 – 3.4
Pairs Intersections

Candidate interactions. Start thinking about what digits can go where. Pointing pairs and naked pairs appear for the first time.

Intermediate

SE 3.4 – 3.8
Hidden triples X-Wings

Subset logic. Multiple candidates must be considered together. You'll start noticing patterns that span rows and columns.

Hard

SE 3.8 – 4.5
Box/line reduction Swordfish

Intersection techniques. Constraints between rows, columns, and boxes create eliminations that require careful candidate tracking.

Expert

SE 4.5 – 5.5
Fish Quads Rectangles Wings

Pattern recognition. X-Wings, Unique Rectangles, XY-Wings, and more. These puzzles demand careful notation and spatial awareness.

MasterSecret

SE 5.5 – 7.0
Wings Chains ALS Medusa

Advanced logic chains. Requires deep reasoning across multiple inference steps. This tier is hidden by default and unlocked via a special code in the app.

ExtremeSecret

SE 7.0 – 11.0
Forcing chains ALS chains Backtracking

Extreme difficulty. May require trial-and-error reasoning or multi-step forcing chains. This tier is hidden by default and unlocked via a special code in the app.

How Rating Works

The engine solves each puzzle using only human-style techniques — no brute force. It tries techniques in order from easiest to hardest, and the SE rating equals the hardest technique needed to reach the solution.

A puzzle that can be solved entirely with hidden singles rates 1.5. A puzzle that needs one X-Wing step rates 3.2, even if every other step is a simple single. The difficulty is always determined by the peak, not the average.

This means puzzles that need forcing chains (SE 7.5+) rate much higher than those solvable with naked singles (SE 2.3), even if both have a similar number of clues.
