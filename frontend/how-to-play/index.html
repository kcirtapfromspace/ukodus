How to Play Sudoku

Fill every row, column, and 3×3 box with the digits 1–9. Each digit appears exactly once in each region. Ukodus generates puzzles with a guaranteed unique solution, powered by a Rust engine running in your browser via WebAssembly.

Getting Started

Click Play and choose a difficulty tier. Five tiers are available by default:

Beginner

Solvable with naked and hidden singles only. A great starting point for new players.

Easy

Introduces locked candidates and basic intersections alongside singles.

Medium

Requires pairs, triples, and pointing/claiming techniques to solve.

Hard

Introduces X-Wings, Swordfish, and other fish patterns alongside quads.

Expert

Demands advanced techniques: XY-Wings, Unique Rectangles, and more.

Secret Tiers

Master and Extreme tiers are hidden by default. Discover how to unlock them for the hardest puzzles.

Controls

Ukodus supports keyboard, mouse, and touch input. Here are the keyboard shortcuts:

Key Action
Arrow keys / h j k l / w a s d Navigate the grid
1–9 Place a digit in the selected cell
0 / Delete / Backspace Clear the selected cell
c Toggle candidate (notes) mode
n Start a new game
u Undo the last move
p Pause the game timer
f Full clear — remove all player entries
x Remove a digit (in candidate mode)
? Get a hint
! Apply the current hint
Enter / Space Confirm selection

Candidate Mode

Press c to toggle candidate mode (also called pencil marks or notes). In this mode, pressing a digit adds or removes it as a small candidate in the selected cell instead of placing it as your answer. Candidates help you track which digits are still possible in each cell—an essential technique for harder puzzles.

When you place a final digit, conflicting candidates in the same row, column, and box are automatically removed.

Hints

Ukodus provides progressive hints that teach you solving techniques rather than just giving answers. Press ? to request a hint:

Step 1: Technique Name

The first press shows which solving technique applies next (e.g., "Naked Pair", "X-Wing"). Try to find it yourself.

Step 2: Involved Cells

Press ? again to highlight the specific cells and candidates involved in the technique.

Step 3: Full Proof

A third press reveals the complete explanation: which candidates can be eliminated and why. Press ! to apply the hint to the board.

Difficulty Tiers

Each puzzle is rated using the Sudoku Explainer (SE) scale based on the hardest technique required to solve it. Ukodus offers 8 tiers, from Beginner (SE 1.5–2.5) through Extreme (SE 9.0+), each requiring progressively more advanced solving techniques.

For a detailed breakdown of every tier, technique requirements, and SE rating ranges, see the Difficulty Tiers page.

Sharing Puzzles

After finishing a game (or during play), click Share to generate an 8-character puzzle code like M1A2B3C4. Share the code with friends, or copy the full URL. The same code works across the web app, iOS app, and terminal UI—identical engine, identical puzzle.

Leaderboard

The leaderboard is fully anonymous. Choose a display tag when you submit your time—no account or sign-up required. Your best times are ranked against other players for each difficulty tier. Move logs are recorded for anti-cheat verification, ensuring fair competition.
